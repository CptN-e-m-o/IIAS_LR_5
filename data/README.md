# Data

Скачайте файл `aa_dataset-tickets-multi-lang-5-2-50-version.csv` из репозитория Hugging Face `Tobi-Bueck/customer-support-tickets`.

После скачивания переименуйте файл в:

```text
customer_support_tickets.csv
```

и положите его в эту папку.

CSV-файл не включен в репозиторий, чтобы не раздувать размер проекта.
